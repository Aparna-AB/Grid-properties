Grid System

CSS Grid Layout is a powerful and flexible system for creating two-dimensional layouts on web pages.

Grid Template columns grid-template-columns: repeat(6, 1fr);

Grid Template columns grid-template-columns: 1fr 2fr 4fr;

Grid Template columns grid-template-columns: 1fr 60px;

Mixed Units fixed(px) and fractional

*Here's how space is divided when you mix units like 1fr and 60px in grid-template-columns: ** Fixed Units Get Priority: The browser first allocates space for any fixed units, like the 60px column in this example. It takes up exactly 60 pixels of width. Fractional Units Claim Remaining Space: The fr units then divide the remaining available space proportionally. In this case, the 1fr column will occupy whatever space is left after the 60px column is accounted for.
